## 🧰 Instalación y uso

### 1. Crear el proyecto con Vite

```bash
npm create vite@latest
````

Selecciona:

* Framework: `React`
* Variante: `JavaScript + SWC`

### 2. Entrar al proyecto e instalar dependencias

```bash
cd nombre-del-proyecto
npm install
```

### 3. Iniciar el servidor de desarrollo

```bash
npm run dev
```
